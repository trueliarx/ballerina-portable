# ballerina portable for windows (automatic download)
ballerina.bat opens a prompt with ballerina immediately available (The env variables are already set).
If not present it automatically downloads jdk and -ballerina- files without installing them in windows, allowing a *portable* use from any folder.

To execute ballerina execute/write+Enter "bal" from the prompt.
